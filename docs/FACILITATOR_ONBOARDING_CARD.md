# Arthamantri Facilitator Onboarding Card (2-Minute Setup)

Use this card when onboarding a participant with low digital confidence.

## English
Participant profile:
- Name: ____________________
- Phone owner: Self / Family / Shared
- Facilitator name: ____________________
- Date: ____________________

2-minute setup checklist:
1. Open Arthamantri and select language.
2. Read pilot consent aloud and record acceptance.
3. Open Money Setup Lite:
   - Select profile (Women-led household / Daily-cashflow worker).
   - Select up to 2 essential goals (Ration, School, Fuel, Medicine, Rent, Recharge, Loan Repayment).
4. Enable permissions:
   - SMS
   - Usage Access
   - Overlay Access
   - Notification Access
5. Tap Start Monitoring.
6. Send one test SMS to verify first alert flow.

Quick safety script to read to participant:
- “This app never asks for OTP, UPI PIN, or bank password.”
- “This app gives safety nudges only.”
- “You can mark alerts useful/not useful anytime.”

Follow-up after 3-5 days:
- Did alerts feel useful? Yes / No
- Were alerts too frequent? Yes / No
- Any confusing message? ____________________

## हिंदी
प्रतिभागी जानकारी:
- नाम: ____________________
- फोन उपयोग: स्वयं / परिवार / साझा
- सहायक का नाम: ____________________
- तिथि: ____________________

2-मिनट सेटअप चेकलिस्ट:
1. Arthamantri खोलें और भाषा चुनें।
2. सहमति संदेश पढ़कर सुनाएं और सहमति दर्ज करें।
3. Money Setup Lite खोलें:
   - प्रोफाइल चुनें (महिला-नेतृत्व वाला परिवार / दैनिक नकदी-प्रवाह कामगार)।
   - अधिकतम 2 आवश्यक लक्ष्य चुनें (राशन, स्कूल, ईंधन, दवा, किराया, रिचार्ज, ऋण भुगतान)।
4. अनुमतियां चालू करें:
   - SMS
   - उपयोग-अनुमति
   - ओवरले अनुमति
   - सूचना अनुमति
5. “निगरानी शुरू करें” दबाएं।
6. एक टेस्ट SMS भेजकर अलर्ट प्रवाह जांचें।

सुरक्षा संदेश (प्रतिभागी को सुनाएं):
- “यह ऐप कभी OTP, UPI PIN या बैंक पासवर्ड नहीं मांगता।”
- “यह ऐप केवल सुरक्षा संकेत देता है।”
- “आप किसी भी अलर्ट को उपयोगी/अनुपयोगी चिह्नित कर सकते हैं।”

3-5 दिन बाद फॉलो-अप:
- अलर्ट उपयोगी लगे? हाँ / नहीं
- अलर्ट बहुत अधिक आए? हाँ / नहीं
- कौन सा संदेश समझने में कठिन था? ____________________
